## SmartUse

A land use mapping project focusing on the greater metropolitan area around Zurich, Switzerland (Metropolitankonferenz ZÃ¼rich).

## Usage

    $ git clone https://gitlab.com/loleg/smartuse.git
    $ cd smartuse
    $ npm install -g yarn
    $ yarn install
    $ cp .env.example .env
    $ yarn start

The app will now be available at http://localhost:3000

The splash page is at http://localhost:3000/splashpage/index.html
